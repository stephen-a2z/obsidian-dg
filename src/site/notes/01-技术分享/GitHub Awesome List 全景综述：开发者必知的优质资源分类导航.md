## 前言

### 如何保持更新

1. **RSS 订阅**：https://www.trackawesomelist.com/rss.xml
2. **邮件订阅**：在 trackawesomelist.com 首页注册 Newsletter
3. **Watch 仓库**：在 GitHub 上 Watch 感兴趣的 awesome 列表，选择 "Releases only" 或 "Custom" 减少噪音
4. **每周浏览**：trackawesomelist.com 提供 [Weekly 视图](https://www.trackawesomelist.com/week/) 汇总一周新增

sindresorhus/awesome 本质上是一个**开发者知识图谱**。无论你是前端工程师、后端架构师、数据科学家、安全研究员还是独立开发者，都能在这里找到对应领域的高质量资源入口。
